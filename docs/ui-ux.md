# UI/UX Guidelines

## Principles

- Mobile first
- Simple
- Fast
- Accessible

## Layout

Ensure:

- Consistent spacing
- Consistent typography
- Clear hierarchy
- Responsive design

## Forms

Must include:

- Validation feedback
- Error states
- Success states

## Accessibility

Check:

- Keyboard navigation
- Focus states
- Color contrast
- ARIA labels where needed

## Pages

Every page should have:

- Loading state
- Empty state
- Error state

## Avoid

- Hidden actions
- Excessive animations
- Layout shifts
- Tiny touch targets
