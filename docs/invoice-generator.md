# Invoice Generator Requirements

## Supported Fields

Seller Details:
- Name
- Address
- GSTIN

Buyer Details:
- Name
- Address
- GSTIN

Invoice:
- Number
- Date

Items:
- Name
- Quantity
- Unit Price
- GST %

## Features

- Dynamic item rows
- Auto calculations
- PDF export
- Print support

## Tax Modes

Support:

- CGST + SGST
- IGST

## Validation

Validate:

- Empty item names
- Negative values
- Invalid GST percentages
- Missing invoice numbers

## Calculation Rules

Always verify:

- Line totals
- GST amount
- Grand total
- Tax splits

Calculation accuracy is more important than UI.
