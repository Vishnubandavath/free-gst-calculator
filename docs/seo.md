# SEO Guidelines

## Primary Goal

Every page should be optimized for organic search traffic.

## Requirements

- Unique title tag
- Unique meta description
- Canonical URL
- Open Graph tags
- Twitter card tags
- Semantic HTML
- Proper heading hierarchy (H1 → H2 → H3)
- Internal linking
- Mobile-friendly design

## Tool Pages

Every calculator page should contain:

- Tool description
- Usage instructions
- FAQ section
- Related tools section

## Blog Articles

- Minimum 1000 words
- One H1 only
- FAQ section
- Internal links
- Relevant schema markup

## Performance Targets

- LCP < 2.5s
- CLS < 0.1
- INP < 200ms

## Avoid

- Duplicate metadata
- Duplicate content
- Multiple H1 tags
- Client-side rendered SEO content
