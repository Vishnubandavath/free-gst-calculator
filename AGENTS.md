# GST Toolkit

## Purpose

This repository contains a GST-focused utility website built with Next.js.

Before starting any task:

1. Read AGENTS.md
2. Read relevant files in /docs
3. Apply those instructions

---

## Product Scope

Current Features:

- GST Calculator
- Reverse GST Calculator
- Invoice Generator
- Blog

Do not add:

- Authentication
- Databases
- OCR
- AI features
- Invoice history
- Customer management

unless explicitly requested.

---

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS 4
- React Hook Form
- jsPDF / pdf-lib
- localStorage
