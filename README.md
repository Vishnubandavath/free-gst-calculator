# VSNEXOS - India's Free Professional GST Calculator

A world-class, fully responsive, ultra-modern GST calculation website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Professional UI/UX**: Modern glassmorphism design with smooth animations using Framer Motion.
- **Accurate GST Engine**: Handles 0%, 3%, 5%, 12%, 18%, and 28% GST rates with CGST/SGST/IGST breakdown.
- **Advanced Calculator**: Multi-item invoice-style table with discounts and CSV export.
- **SEO Optimized**: Dynamic metadata, sitemap.xml, robots.txt, and JSON-LD schema.
- **AdSense Ready**: Comprehensive content pages (Guide, FAQ, About, Contact) and legal pages.
- **Performance Focused**: Static export compatible with high Lighthouse scores.
- **Mobile First**: Touch-optimized inputs and responsive layouts.
- **Dark Mode**: Native support for dark and light themes.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide Icons
- **Language**: TypeScript
- **Deployment**: Static hosting compatible (Vercel, Netlify, Cloudflare Pages)

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Deployment

The project is configured for static export. The output will be in the `out` directory.

```bash
npm run build
```

Upload the contents of the `out` folder to any static hosting provider.
